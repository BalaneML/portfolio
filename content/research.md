# 研究

## 研究テーマ
生成AIを用いた人口合成手法の開発

## 研究の概要・問い
都市交通計画や政策立案には、個人単位の行動データである「人口マイクロデータ」が不可欠です。しかし実際の調査票情報は秘匿処理が施されており、研究者が直接アクセスすることには大きな制約があります。そこで本研究では、実データの統計的性質を保ちながら、プライバシーリスクを低減した合成データを生成する手法に取り組んでいます。あわせて、特定の属性の組み合わせが調査データに存在しないために推計が困難になる「サンプリングゼロ問題」や、合成した個人に一日の活動スケジュールを割り付ける「活動割り付け」にも挑戦しています。

## 社会的・学術的意義
個人の移動・行動データは、都市計画・交通予測・災害管理・疫学予測など幅広い分野で重視されています。一方で、交通・都市計画で扱う表形式データには、連続変数と離散変数の混在・連続変数の多峰性分布・カテゴリ変数の著しい不均衡といった固有の難しさがあり、画像や言語向けの既存手法をそのまま適用することはできません。プライバシーを守りつつ妥当なデータを供給できれば、データ制約に阻まれていた数多くの研究・政策立案を前進させられます。

## 独自のアプローチ
生成AIモデルとパーソントリップ調査などの実活動データを組み合わせ、人口合成と活動割り付けの両課題を一貫して扱う点が特徴です。ルールベース手法の画一性と、深層学習手法における活動の時間的整合性の欠如 — その双方の弱点を、統計手法（IPF）とVAEの融合などによって乗り越えることを目指しています。

## 研究スタイル・ツール

### 手法1
**名称**: 深層生成モデル（VAE / GAN / CTGAN / TVAE）
**内容**: 論文の理論を精読したうえで、連続値の多峰性正規化やカテゴリ変数の不均衡といった表形式データ特有の課題まで踏み込んで理解し、実装に反映しています。

### 手法2
**名称**: Python / PyTorch / NumPy
**内容**: モデルを自分の手で実装し、その挙動を一つ一つ検証。「動くこと」だけでなく「なぜそう動くのか」を確かめながら研究を進めています。

### 手法3
**名称**: 統計手法 × 実データ（IPF / パーソントリップ調査）
**内容**: 機械学習の理論・統計手法・ドメイン知識を統合し、実際の活動データと組み合わせることで、生成データの妥当性とプライバシー保護を両立させます。

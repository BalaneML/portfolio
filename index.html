過去 · 現在 · 未来

過去

専門学校から、徳島大学、そして大阪大学へ — 環境を自ら変えてきた

情報システムの基礎を専門学校で学んだ後、より深く学びたいという思いから徳島大学へ編入し、深層学習と画像処理に取り組みました。 学部3年次後半、仮配属していた研究室が教員の都合で変わり、研究テーマを白紙から考え直すことに。 私はこれを「自分のやりたい研究に挑戦できる絶好の機会」と捉え直し、関心分野の論文を幅広く調べて方向性を定め、新しい担当教員へ自ら提案。 結果として、他学部との共同研究で「物体検出モデルを用いた豚の交配行動自動検出システム」というテーマを獲得し、立ち上げから実装・評価まで主体的に進めました。

現在

大阪大学大学院で、生成AIによる人口合成を研究

現在は大阪大学大学院で、生成AIを用いた人口合成手法の開発に取り組んでいます。 VAE・GAN・CTGAN・TVAEといった深層生成モデルを、論文の理論を精読したうえでPyTorchやNumPyで自ら実装し、その挙動を一つ一つ検証。 プライバシーを保護しながら、現実の人々の行動を統計的に妥当に再現する「合成データ」を生み出すことに挑んでいます。 技術の形骸化を防ぐため、シンポジウムや技術カンファレンスにも能動的に足を運び、最新の知見を吸収し続けています。

未来

研究の成果を、社会とコミュニティへ還元する

目指すのは、自分が扱う手法を一つの応用にとどめず、業界の基盤となるような汎用的なパイプラインへと育てることです。 将来は研究プロジェクトをリードする立場で、基礎研究と社会実装のバランスを取りながら、学術的価値と社会的価値の両立に貢献したい。 博士課程への進学も視野に入れつつ、グローバルな環境で、自分の成果が社会やコミュニティに広く還元される研究開発に携わることを目標としています。

私が大切にしているのは、主体的に動き、挑戦を重ねること、そして理論を理解したうえで自分の手で実装に落とし込むことです。 専門学校から徳島大学、そして大阪大学へ — その都度、与えられた環境で待つのではなく、自ら機会を切り拓き、仲間とともに切磋琢磨してきました。 研究で最も面白いと感じるのは、課題を明確にし、仮説を立てる創造的な瞬間です。そしてその難しさは、いつも担当教員や先輩・同期・後輩といった「頼れる人たち」との対話を通じて乗り越えてきました。 深く考え抜く力と、人と協働する力。この二つを軸に、技術を通じて社会に意味のある価値を生み出していきたいと考えています。

研究

テーマと意義 · 研究スタイル · ツール

生成AIを用いた人口合成手法の開発

都市交通計画や政策立案には、個人単位の行動データである「人口マイクロデータ」が不可欠です。 しかし実際の調査票情報は秘匿処理が施されており、研究者が直接アクセスすることには大きな制約があります。 そこで本研究では、実データの統計的性質を保ちながら、プライバシーリスクを低減した合成データを生成する手法に取り組んでいます。 あわせて、特定の属性の組み合わせが調査データに存在しないために推計が困難になる「サンプリングゼロ問題」や、合成した個人に一日の活動スケジュールを割り付ける「活動割り付け」にも挑戦しています。

個人の移動・行動データは、都市計画・交通予測・災害管理・疫学予測など幅広い分野で重視されています。 一方で、交通・都市計画で扱う表形式データには、連続変数と離散変数の混在・連続変数の多峰性分布・カテゴリ変数の著しい不均衡といった固有の難しさがあり、画像や言語向けの既存手法をそのまま適用することはできません。

私のアプローチは、生成AIモデルとパーソントリップ調査などの実活動データを組み合わせ、人口合成と活動割り付けの両課題を一貫して扱うことです。 ルールベース手法の画一性と、深層学習手法における活動の時間的整合性の欠如 — その双方の弱点を、統計手法（IPF）とVAEの融合などによって乗り越えることを目指しています。

研究スタイル・使用ツール

📚

深層生成モデル（VAE / GAN / CTGAN / TVAE）

論文の理論を精読したうえで、連続値の多峰性正規化やカテゴリ変数の不均衡といった表形式データ特有の課題まで踏み込んで理解し、実装に反映しています。

💻

Python / PyTorch / NumPy

モデルを自分の手で実装し、その挙動を一つ一つ検証。「動くこと」だけでなく「なぜそう動くのか」を確かめながら研究を進めています。

🔬

統計手法 × 実データ（IPF / パーソントリップ調査）

機械学習の理論・統計手法・ドメイン知識を統合し、実際の活動データと組み合わせることで、生成データの妥当性とプライバシー保護を両立させます。
